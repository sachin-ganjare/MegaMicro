How to Add Internet Speed Meter | MegaMicro Solutions

You can use **Traffic Monitor** to display the live network speed on either the taskbar or as a floating gadget on the desktop in Windows 11. Traffic Monitor is a free and open-source app that works perfectly fine on Windows 11 ¹. Here are the steps to install and add its net speed monitor to the Windows 11 taskbar: 1. Go to [GitHub](https://github.com/zhongyang219/TrafficMonitor/releases) to download Traffic Monitor. 2. Check the latest release and select the x64.zip under “Assets” to download it. 3. Once downloaded, unzip the file and open the TrafficMonitor.exe from the extracted folder. 4. The app will start off with a floating network speed monitor gadget on the desktop which you can move around easily. 5. To enable it to show on the taskbar instead, right-click the floating network meter and select “Show Taskbar Window”. 6. The net speed monitor will now be displayed on the Windows 11 taskbar. Alternatively, you can use **Net Speed Meter** to display the real-time network or internet speed directly on the taskbar. Net Speed Meter is available on Microsoft Store ². Here are the steps to install and enable it: 1. Open Microsoft Store via Start menu. 2. Search for “Net Speed Meter” using the search bar in the Microsoft Store’s window. 3. Select Net Speed Meter and click Get to install the app. 4. Once the installation is completed, click Open to launch the app. 5. The app will start out as a floating network meter on the desktop. 6. You’ll be asked here whether you want to keep Net Speed Meter on your taskbar, click Yes to confirm. 7. Finally, Net Speed Meter is successfully enabled in Windows 11.
